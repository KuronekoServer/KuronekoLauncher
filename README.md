### Getting Started

**System Requirements**

* [Node.js][nodejs] v16

---

**Clone and Install Dependencies**

```console
> git clone https://github.com/KuronekoServer/KuronekoLauncher.git
> cd HeliosLauncher
> npm install
```

---

**Launch Application**

```console
> npm start
```
